# Shopping List Telegram Bot

Telegram-бот для совместного списка покупок на Python, aiogram и SQLite.

## Возможности

- Совместные списки покупок по коду приглашения.
- Хранение данных в SQLite.
- Токен бота через переменные окружения.
- Добавление одного товара или нескольких товаров сразу.
- Inline-кнопки для отметки товара купленным.
- Удаление товара кнопкой рядом с ним.
- История удаленных и очищенных товаров.
- Очистка текущего списка.

## Команды

- `/start` - показать справку.
- `/newlist` - создать новый список.
- `/join код` - присоединиться к существующему списку.
- `/mycode` - показать код текущего списка.
- `/add название количество единица` - добавить товар.
- `/addmany` - добавить несколько товаров, каждый с новой строки.
- `/show` - показать список с кнопками.
- `/remove название` - удалить товар по названию.
- `/history` - показать историю удаленных товаров.
- `/clear` - очистить текущий список.

## Запуск локально

1. Создать виртуальное окружение:

```bash
python -m venv .venv
```

2. Активировать окружение:

```bash
.venv\Scripts\activate
```

3. Установить зависимости:

```bash
pip install -r requirements.txt
```

4. Создать `.env` по примеру `.env.example`:

```env
BOT_TOKEN=your_bot_token_here
```

Если нужен прокси:

```env
BOT_PROXY=http://host:port
```

5. Запустить бота:

```bash
python bot.py
```

## Данные

Бот хранит данные в локальной базе `shopping_list.db`.

Файлы `.env`, `shopping_list.db` и `data.json` не должны попадать в GitHub.
